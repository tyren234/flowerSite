# flowerSite
Personal react website

[https://tyren234.github.io/flowerSite/](https://tyren234.github.io/flowerSite/)

# Develop
In `./flower-site-ui` run
```bash
npm i
```
To run frontend run
```
npm run start
```


# Update
To update run 

```bash
npm run build
```

in `./flower-site-ui` and copy new files to `./`
